# FP&A SKILL SYSTEM

## What this is
A modular, AI-executable FP&A system that encodes how top finance teams operate.

## What this is NOT
- Not a collection of prompts
- Not a static finance guide

## Core Idea
Turn finance expertise into:
→ reusable SKILL modules  
→ orchestrated workflows  
→ decision-making systems  

## How to Use

### Step 1: Define the problem
Example:
"Why is EBITDA below forecast?"

### Step 2: Trigger workflow
→ variance_analysis  
→ forecasting  
→ scenario_planning  

### Step 3: Generate output
→ executive-ready recommendation  

## System Layers
- SKILLS → atomic capabilities
- PLAYBOOKS → multi-step workflows
- AGENTS → role-based orchestration
